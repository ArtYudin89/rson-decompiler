# Reference fixtures

Эталоны интерактивного SCR-декомпила самого редактора (RScript) — «золотой
источник» для сверки раскладки полотна. Приватные тестовые данные.

| Файл | Что это |
|---|---|
| `<Name>.scr` | исходный бинарник (вход декомпилятора) |
| `<Name>_recovery.rson` | результат интерактивного декомпила редактора — ground truth (шапка, поля, порядок ключей, координаты) |
| `<Name>_recovery.png` | скрин полотна редактора для recovery-файла (целевой вид) |
| `<Name>_mine.png` | скрин полотна для нашего вывода (для сравнения) |

Сверка нашего вывода с эталоном:

```bash
python run.py reference/Coalition_Heart.scr        # -> reference/Coalition_Heart.rson
# затем сравнить состав узлов/поля/координаты с Coalition_Heart_recovery.rson
```

`Coalition_Heart` — маленький (совпадает с recovery по составу/полям и пиксельно
по колонке объектов). `Mod_ShuKlissan` — крупный (много групп/состояний/диалогов;
показывает пределы приближения раскладки).
